Ce que je fais

Développement Front-end

Développer une interface web claire, fluide et ergonomique en utilisant les langages HTML, CSS et Javascript. Framework utilisés: AngularJS, ReactJS, NextJS, Bootstrap.

Développement Back-end

Mettre en place la programmation au niveau du serveur afin d'héberger toutes les informations et les codes, mais aussi savoir gérer les bases de données pour exécuter les requêtes qui sont réalisées sur le site web par les internautes. Langages utilisés: PHP, Java, Python, MySQL.

Responsiv design

Technique de conception d’interface digitale qui fait en sorte que l’affichage d’une quelconque page d’un site s’adapte de façon automatique à la taille de l’écran du terminal qui le lit.

Clean code

Le clean code désigne une série de principes pour produire un code compréhensible de façon intuitive et facile à modifier. Compréhensible signifie dans ce cas un code immédiatement intelligible par n’importe quel développeur.

Référencement naturel

Le référencement naturel ou SEO rassemble l’ensemble des actions qui permettent d’améliorer le positionnement d’un site sur les moteurs de recherche et donc sa visibilité.

Cybersécurité

La cybersécurité est la pratique consistant à protéger les systèmes, les réseaux et les programmes contre les attaques numériques. Ces cyberattaques visent généralement à accéder à des informations sensibles, à les modifier ou à les détruire.
